Binding function on two-way-databinding

Written by Martin Knopf

The binding function used in e2e tests doesn’t work as expected on a two-way-databinding.

Calling binding() on a two-way-databinding (like ng-model="binding-name") throws an error message that says Binding selector 'binding-name' did not match.. I don’t know the reason. Maybe I’ll get an explanation on my stackoverflow question.
